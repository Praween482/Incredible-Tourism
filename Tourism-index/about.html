About this project

This project was created by Class 10th A for the school assignment "Incredible Tourism: Heritage, Hospitality & Tourism". It demonstrates how heritage sites, modern hospitality and tourism infrastructure form visitor experiences across North India.

Project summary

A multi-page interactive site with cinematic visuals, searchable and clickable place cards, and an embedded Google Map to explore locations. The 3D physical model at the exhibition can link via QR codes to the map for live navigation and “Learn More” details.

How to use

  • Open Map and click a place to open its location.
  • From the Home or States page click a card to open that place on the Map.
  • For a full demo, host the folder on a local server or GitHub Pages (I can help with that).
